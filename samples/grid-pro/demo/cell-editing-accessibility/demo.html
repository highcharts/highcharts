Shows how editable cells work with accessibility support in Highcharts Grid Pro. Double-click an editable cell or press `Enter `to start editing, then try typing a non-number in the `stock` or `price` columns to hear the validation announcement. For other accessibility features, see also the Grid Lite accessibility demo. For implementation guidance and additional details, refer to the accessibility documentation.
