# Healthcare Demos
This demos contains demos for the Highcharts for Healthcare page. They all use the same demo.html file and load the chart specified by the "chart" URL parameter (e.g. chart=accessibility)

- [Hero - Bubble Chart](http://localhost:3030/samples/view?path=highcharts/website/industry-healthcare-demos&chart=hero)
- [Advanced accessibility](http://localhost:3030/samples/view?path=highcharts/website/industry-healthcare-demos&chart=accessibility)
- [Live CSV data](http://localhost:3030/samples/view?path=highcharts/website/industry-healthcare-demos&chart=csv)
- [Polygon Series](http://localhost:3030/samples/view?path=highcharts/website/industry-healthcare-demos&chart=polygon)
- [Scatter with linear regression](http://localhost:3030/samples/view?path=highcharts/website/industry-healthcare-demos&chart=scatter)
- [Large heatmap](http://localhost:3030/samples/view?path=highcharts/website/industry-healthcare-demos&chart=heatmap)
- [Pictorial Series](http://localhost:3030/samples/view?path=highcharts/website/industry-healthcare-demos&chart=pictorial)
