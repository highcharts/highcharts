# Finance Demos
This demos contains demos for the Highcharts for Finance page. They all use the same demo.html file and load the chart specified by the "chart" URL parameter (e.g. chart=exclusiveSeries)

- [Hero - All Indicators](http://localhost:3030/samples/view?path=highcharts/website/industry-finance-demos&chart=hero) 
- [Exclusive Series](http://localhost:3030/samples/view?path=highcharts/website/industry-finance-demos&chart=exclusiveSeries)    
- [Stock GUI](http://localhost:3030/samples/view?path=highcharts/website/industry-finance-demos&chart=stockGui)  
- [Two Standalone Navigators](http://localhost:3030/samples/view?path=highcharts/website/industry-finance-demos&chart=navigators)  
- [MACD with pivot points](http://localhost:3030/samples/view?path=highcharts/website/industry-finance-demos&chart=macd)  
- [Boost module](http://localhost:3030/samples/view?path=highcharts/website/industry-finance-demos&chart=boost)  
- [Customization](http://localhost:3030/samples/view?path=highcharts/website/industry-finance-demos&chart=customization)
