# Software Demos
This demos contains demos for the Highcharts for Software page. They all use the same demo.html file and load the chart specified by the "chart" URL parameter (e.g. chart=treempa)

- [React Bubble Chart](http://localhost:3030/samples/view?path=highcharts/website/industry-software-demos&chart=react)  
- [Area Chart Race](http://localhost:3030/samples/view?path=highcharts/website/industry-software-demos&chart=areaRace)  
- [Live Data](http://localhost:3030/samples/view?path=highcharts/website/industry-software-demos&chart=liveData)  
- [A11y](http://localhost:3030/samples/view?path=highcharts/website/industry-software-demos&chart=a11y)
