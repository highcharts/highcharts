Benchmark stage

This study compares two Grid rendering profiles on the same generated dataset: one with row virtualization enabled and one with a full DOM render. Each run recreates the grid from scratch, measures initialization until the async Grid promise resolves and the next paint lands, then runs a fast synthetic scroll and a resize/reflow loop. Use it as a repeatable baseline when validating rendering optimizations.
