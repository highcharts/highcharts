# Changelog for Highcharts Dashboards v4.2.0 (2026-05-12)

- Restored cell `width`/`height` options.

## Bug fixes
- Fixed [#24446](https://github.com/highcharts/highcharts/issues/24446), changing the connector in edit mode did not update the Grid Component properly.
- Fixed double re-render when polling is activated.
- Fixed component namespaces in Dashboards.
- Fixed lacking dts for minified module src files.
- Fixed [#24258](https://github.com/highcharts/highcharts/issues/24258), changed event type for cell.destroy.
- Fixed [#24086](https://github.com/highcharts/highcharts/issues/24086), the grid was not destroyed properly on board.update() call.
- Fixed [#24067](https://github.com/highcharts/highcharts/issues/24067), dataTable was not saved in gridOptions on update.
