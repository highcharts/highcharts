# Changelog for Highcharts Dashboards v4.1.0 (2026-01-21)

- Added basic `board.update()` method.
- Added `update` argument to DataPool's `setConnectorOptions` method. See [#23935](https://github.com/highcharts/highcharts/issues/23935).
- Added `update` method to Data Connectors. See [#21583](https://github.com/highcharts/highcharts/issues/21583).
- Added support for setting global options with `Dashboards.setOptions()`.
- Added the possibility to full-screen individual dashboard components, see [#23547](https://github.com/highcharts/highcharts/issues/23547).

## Bug fixes
- Fixed [#23952](https://github.com/highcharts/highcharts/issues/23952), wrong reference to nestedLayout.
- Fixed [#23948](https://github.com/highcharts/highcharts/issues/23948), row and cell deleting events were not triggered.
- Fixed [#23862](https://github.com/highcharts/highcharts/issues/23862), highlight sync was incorrect when the connector was used with multiple data tables.
- Fixed [#23859](https://github.com/highcharts/highcharts/issues/23859), the `metadata` was not cloned in the data table.
- Fixed [#23699](https://github.com/highcharts/highcharts/issues/23699), error in the console when the `toggle` editable option was added.
