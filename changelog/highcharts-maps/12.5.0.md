# Changelog for Highcharts Maps v12.5.0 (2026-01-12)

- Most changes listed under Highcharts 12.5.0 above also apply to Highcharts Maps 12.5.0.
- Added better default `tooltip.pointFormat` to the `MapPoint` series. Closes [#23883](https://github.com/highcharts/highcharts/issues/23883).

## Bug fixes
- Fixed [#23019](https://github.com/highcharts/highcharts/issues/23019), a regression in v12.2.0 causing `nullInteraction` to break in maps.
